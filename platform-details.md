# عقار إنسايت — توثيق شامل للمنصة

---

## 1. نظرة عامة

**اسم المنصة:** عقار إنسايت (Aqar Insight)  
**الوصف:** منصة عقارية سعودية متكاملة تجمع بين سوق العقارات الحي، والتحليلات الذكية، ودليل المسوّقين ومزودي الخدمات، وإدارة طلبات العملاء.  
**اللغة:** عربية بالكامل (RTL)  
**التقنيات:** React + Vite (الواجهة) / Express 5 (الخادم) / PostgreSQL + Drizzle ORM (قاعدة البيانات)  
**الخط:** Cairo (Google Fonts)

## 6. نظام الإعلانات العقارية

### 6.1 حقول الإعلان الأساسية
| الحقل | النوع | الوصف |
|-------|-------|-------|
| العنوان | نص | عنوان الإعلان |
| الوصف | نص | وصف تفصيلي |

### 6.2 الموقع الجغرافي (4 مستويات)
| المستوى | الحقل | مثال |
|---------|-------|------|
| المنطقة | region | الرياض |
| المحافظة/المدينة | city | الرياض |
| المركز | markaz | — |
| الحي | district | حي الياسمين |

### 6.3 مواصفات العقار
- **غرف النوم** - **دورات المياه** - **المجالس** - **المطابخ** - **غرف الخادمة/السائق**
- **الطابق** - **عدد الطوابق** - **عمر العقار** - **حالة الأثاث** - **عرض الشارع** - **الواجهة**

### 6.4 المرافق والخدمات (50+ خيار)
**داخلية:** مسبح، جيم، جاكوزي، ملحق، تدفئة، سخان شمسي، خزان ماء، مولد كهرباء، طاقة شمسية، غاز طبيعي، فلتر ماء، مجلس، غرفة صلاة، غرفة ملابس، مصعد، مواقف سيارات، حديقة، سطح، بلكونة، قبو، تكييف، واي فاي، كاميرات أمان، منزل ذكي

**قريب من:** مدارس، مستشفيات، مساجد، مولات، مواصلات، حدائق، طرق رئيسية، صيدليات، بنوك، مطاعم، حضانات، نوادي رياضية، محطات وقود، جامعات

### 6.5 الوثائق والحالة
- **حالة الصك** - **حالة الترخيص** - **رقم المرجع** - **صلاحية الرهن**
- **تاريخ الإتاحة** - **قابل للتفاوض** - **بيع مباشر من المالك**

### 6.6 شارات الإعلان
- 🏆 **مميّز** (featured) - **عاجل** (urgent) - **حصري** (exclusive) - **موثّق** (verified)

### 6.7 الوسائط
- **صور متعددة** (رفع مباشر إلى التخزين السحابي)
- **رابط فيديو** - **مخطط الطابق** - **جولة افتراضية**

## 7. نظام التحليلات

### 7.1 التحليلات الحية (من جدول listings)
يحتوي على **7 تبويبات:**

| التبويب | الوصف |
|---------|-------|
| نظرة عامة | KPIs رئيسية: عدد العقارات، متوسط السعر، سعر المتر، جديد آخر 7/30 يوم |
| التوزيع الجغرافي | مقارنة الأسعار بين المناطق والمدن |
| أنواع العقارات | توزيع العقارات حسب النوع مع متوسط السعر |
| اتجاهات الأسعار | رسوم بيانية للأسعار عبر الزمن |
| مؤشر السوق | مؤشر صحة السوق (0-100) + عرض وطلب |
| تحليل الاستثمار | معدل الدوران، كثافة العرض، تشتت الأسعار |
| التحليل الذكي | رؤى تلقائية مولّدة من البيانات |

### 7.2 التحليلات التاريخية (من جدول properties)
- مقارنة الأسعار السنوية - اتجاهات الأسعار - توزيع أنواع العقارات - مقارنة الأحياء

### 7.3 تقييم السعر (Benchmark)
في صفحة تفاصيل كل عقار: يُقارن سعر العقار بمتوسط السوق في نفس المنطقة ونوع العقار ويُظهر إن كان السعر **مناسب / مرتفع / منخفض**.

## 13. نظام المصادقة والأمان

### 13.1 تسجيل الدخول
- يقبل: اسم المستخدم **أو** البريد الإلكتروني
- حماية بـ bcrypt (12 rounds)
- الجلسة محفوظة في قاعدة البيانات (connect-pg-simple)

### 13.2 استعادة كلمة المرور
1. المستخدم يدخل بريده → يصل رابط إعادة تعيين عبر SMTP
2. الرابط صالح لمدة محدودة (token مشفّر)
3. بعد الاستخدام لا يعمل الرابط مرة أخرى

### 13.3 حماية الطلبات
- Rate limiting على طلبات المصادقة
- التحقق من صحة البيانات قبل الحفظ
- جلسة مشتركة بين الويب والتطبيق

## 15. نظام تحديد الموقع (الجغرافيا)

### 15.1 التسلسل الهرمي الجغرافي السعودي
**منطقة → محافظة → مركز → حي** (13 منطقة سعودية كاملة)

### 15.2 القوائم المنسدلة المتتالية
عند اختيار المنطقة تظهر محافظاتها، ثم مراكزها، ثم أحياؤها تلقائياً.

### 15.3 الخريطة التفاعلية
- مكتبة **Leaflet** مع **MarkerCluster**
- تجميع الإعلانات عند التكبير
- ألوان مختلفة للبيع والإيجار
- نقر على الدبوس يفتح ملخص الإعلان

### 15.4 تحديد الموقع في النماذج
- مكوّن `LocationPicker` يسمح بتحديد الموقع على الخريطة
- دعم **Geolocation** (تحديد الموقع الحالي)
- **Reverse Geocoding** عبر Nominatim
- التحقق من أن الموقع داخل حدود المملكة العربية السعودية
