### مستند التصميم الشامل للتطبيق المحمول — بناءً على منصة الويب كمرجع أساسي

## القسم الأول: استخراج الميزات الكاملة من منصة الويب

### 1.1 هيكل الصفحة الرئيسية (Homepage)

**شريط التنقل العلوي:**
- الشعار وعلامة "عقار إنسايت" يساراً
- شريط بحث مركزي
- أزرار تسجيل الدخول / إنشاء حساب / زر "أضف إعلانك"
- مبدّل اللغة عربي/إنجليزي

**قسم الهيرو (Hero Section):**
- خلفية صورة بانورامية لمدينة سعودية (فيلتر معتم)
- عنوان رئيسي: "خذ القرار الصح في عقارك"
- شريط بحث متقدم مدمج يحتوي على:
  - منطقة إدارية → محافظة → حي (cascading dropdowns)
  - نوع العقار (dropdown مجمّع بفئات)
  - نوع الصفقة: بيع / إيجار / إيجار شهري / استثمار / مزاد
  - زر البحث
- إحصائية متحركة صغيرة: "أعلى 5 مناطق · متوسط سعر م²"
- زران ثانويان: "تصفح جميع العقارات" + "أضف إعلانك الآن"

**شريط الإحصائيات السريعة (KPI Strip):**
- إجمالي الإعلانات النشطة
- متوسط سعر المتر المربع
- عدد الإعلانات للبيع
- عدد للإيجار
- إعلانات هذا الشهر

**تصفح حسب النوع (Category Chips):**
10 فئات سريعة: شقق، فلل، أراضي، عمارات، مكاتب، محلات، استوديو، دوبلكس، مستودعات، مزارع

**العقارات المميزة (Featured Listings):**
- كاروسيل أفقي لبطاقات العقارات المميزة
- بطاقة عقار تشمل: صورة، سعر، نوع، موقع، غرف، حمامات، مساحة، شارة "مميز"

**الخريطة التفاعلية (Interactive Map):**
- خريطة Leaflet كاملة مدمجة في الصفحة الرئيسية
- دبابيس ملوّنة حسب الحالة مع clustering
- بطاقة معاينة عند الضغط على الدبوس
- تكامل مع نظام الفلاتر

**مؤشرات السوق (Market Indicators):**
- مؤشر "حالة السوق" (نشط جداً / قوي / متوازن / هادئ) بنقطة ملونة
- رسم بياني خطي لتطور الأسعار بمرور الوقت
- مؤشر دائري لتوزيع أنواع العقارات
- مؤشر الطلب في مقابل العرض

**السوق الخدمي (Services Marketplace):**
- عرض لأحدث مزودي الخدمات العقارية (4 بطاقات)
- تصنيفات: بناء وتشييد، تشطيبات، كهرباء ومياه، تكييف...

**سوق الطلبات (Requests Marketplace):**
- عرض لآخر طلبات العملاء (3 بطاقات)
- أنواع: طلب عقار / طلب خدمة / طلب مسوّق

**قسم "لماذا نحن":**
- بيانات موثّقة، خريطة تفاعلية، تحليلات السوق، مسوّقون معتمدون

**تقييم المنصة (Platform Rating Widget):**
- متوسط التقييم من المستخدمين بنجوم
- زر لإضافة تقييم جديد

### 1.2 صفحة قوائم العقارات (Listings)

**شريط البحث والفلترة:**
- مربع بحث نصي (عنوان، حي، مدينة)
- زر "الفلاتر" مع عداد الفلاتر الفعّالة
- فلاتر سريعة أفقية: الكل / للبيع / للإيجار / استثمار

**الفلاتر المتقدمة (Advanced Filters Panel):**
- **الموقع:** منطقة → محافظة (cascading) → حي (free text)
- **نوع العقار:** grouped بفئات (سكني: شقة، فيلا، دوبلكس... | تجاري: محل، مكتب... | أراضي | زراعي...)
- **نوع الصفقة:** بيع / إيجار / إيجار شهري / استثمار / مزاد
- **السعر:** حد أدنى / حد أقصى (ريال سعودي)
- **المساحة:** حد أدنى / حد أقصى (م²)
- **عدد غرف النوم:** أي / 1 / 2 / 3 / 4 / 5+
- **عدد دورات المياه:** أي / 1 / 2 / 3 / 4+
- **الترتيب:** الأحدث / الأعلى سعراً / الأقل سعراً / الأكبر مساحةً

**عرض النتائج:**
- شبكة 3 أعمدة (ويب) → 1 أو 2 عمود (جوال)
- عداد النتائج: "عثر على X عقار"
- ترقيم الصفحات (pagination)
- زر تحديث

---

### 1.3 صفحة تفاصيل العقار (Listing Detail)

**معرض الصور:**
- صورة رئيسية كبيرة (3/4 من العرض) + صورتان جانبيتان
- عداد الصور (1 من X)
- زر "عرض كل الصور" يفتح معرض كامل الشاشة
- زر مشاهدة الفيديو (إن وُجد)
- دوران ناعم بين الصور

**رأس التفاصيل:**
- عنوان العقار (h1)
- شارات: نوع الصفقة (للبيع / للإيجار) + الحالة (نشط / مباع / مؤجر / محجوز) + مميز + عاجل + حصري
- السعر الرئيسي + سعر المتر المربع
- شارة التحقق (موثّق) إن وُجدت

**الموقع:**
- مسار التنقل: منطقة ← محافظة ← حي
- رقائق الموقع (Location Chips) بأيقونات

**شبكة التفاصيل التقنية (Facts Grid):**
- المساحة الإجمالية (م²)
- عدد غرف النوم
- عدد دورات المياه
- عدد الأدوار / الطابق
- نوع الصك
- اتجاه الواجهة (شمالي / جنوبي...)
- التشطيب (كامل / شبه مشطّب / هيكل...)
- الأثاث (مفروش / غير مفروش)
- جودة البناء (فاخر / ممتاز / جيد...)
- غرفة السائق / غرفة الخادمة
- سنة البناء
- الغرض (سكني / تجاري / صناعي...)

**مزايا العقار (Amenities Grid):**
موقف سيارات، مصعد، حديقة، سطح، مسبح، غرفة مصلى، تكييف مركزي، إنترنت، نظام أمني، منزل ذكي، مستودع تخزين، شرفة، قبو

**الخدمات القريبة (Nearby Services):**
مدارس، مستشفيات، مساجد، مراكز تسوق، مواصلات عامة، حدائق، طرق رئيسية

**وصف العقار:**
نص حر قابل للتوسيع "عرض المزيد"

**خريطة الموقع:**
- خريطة Leaflet مدمجة بارتفاع 280px
- دبوس قابل للنقر يفتح خرائط جوجل / Apple Maps

**تقييم السعر (Price Benchmark):**
- مؤشر النسبة المئوية: "سعرك أقل من متوسط السوق بـ X%"
- مقارنة: سعرك / متوسط السوق / أدنى سعر / أعلى سعر
- وسمة: فرصة / مناسب / مرتفع

**المسوّق / البائع:**
- صورة المسوّق
- الاسم، اسم المكتب، شارة "موثّق"
- أزرار: اتصال / واتساب / بريد إلكتروني

**عقارات مشابهة:**
- قائمة أفقية لـ 4-6 عقارات مشابهة

**شريط الإجراءات السفلي (للمالك):**
- تعديل / حذف / تغيير الحالة (مباع / مؤجر / أرشفة)
- الإبلاغ عن الإعلان

### 1.4 الخريطة التفاعلية (Map Page)

- خريطة Leaflet كاملة الشاشة
- دبابيس لونية بشارة السعر بالريال
- Clustering تلقائي عند التكبير/التصغير
- بطاقة معاينة عند الضغط: صورة + سعر + عنوان + غرف
- فلاتر على الخريطة: منطقة / المدينة / نوع العقار
- زر "قائمة" / "خريطة" للتبديل
- تحديد الموقع الحالي

---

### 1.5 التحليلات ومؤشرات السوق (Analytics)

**5 تبويبات تحليلية:**

**A — مؤشر السوق:**
- درجة السوق (0-100) مع مؤشر بصري (gauge)
- اتجاه السوق: صاعد / هابط / مستقر
- مستوى النشاط: نشط جداً / متوسط / هادئ
- إجمالي عدد الإعلانات النشطة
- إعلانات جديدة آخر 7 أيام / 30 يوماً
- نسبة التشغيل (Turnover Rate)

**B — الأسعار:**
- رسم خطي لتطور متوسط الأسعار (يومي/أسبوعي/شهري/ربع سنوي/سنوي)
- رسم خطي لتطور سعر المتر المربع
- البيانات: توزيع الأسعار بين البيع والإيجار والاستثمار
- أعلى سعر / أقل سعر / متوسط / وسيط / الربع الأول / الثالث

**C — العرض والطلب:**
- رسم دائري (Pie Chart) لتوزيع الإعلانات حسب نوع العقار مع نسب مئوية
- جدول مقارنة: بيع vs إيجار vs استثمار (عدد + متوسط سعر)
- مخطط أعمدة لأكثر المناطق نشاطاً

**D — الأحياء:**
- قائمة الأحياء مرتبةً حسب: عدد الإعلانات / متوسط السعر / سعر المتر
- فلتر بالمنطقة والمحافظة ونوع الإعلان
- شارة ذكية لكل حي: "فرصة" / "متوازن" / "مرتفع"

**E — الذكاء التحليلي:**
- رؤى ذكية نصية مُولَّدة من البيانات (Smart Insights)
- مثال: "حي النرجس في الرياض يسجل أعلى نشاط هذا الأسبوع"
- توصيات استثمارية

---

### 1.6 مقارنة الأحياء (Districts)

- خريطة حرارية (Heatmap) تلوّن الأحياء حسب سعر المتر:
  - أخضر: أسعار منخفضة / أصفر: متوسطة / برتقالي: مرتفعة / أحمر: مرتفعة جداً
- مفتاح الألوان (Legend)
- فلتر بالمنطقة / نوع الإعلان
- قائمة الأحياء بمعلومات تفصيلية: عدد الإعلانات + متوسط السعر + سعر المتر + شارة التصنيف
- إمكانية اختيار حيّين للمقارنة جنباً إلى جنب:
  - مخطط الأعمدة المقارن
  - الفرق في المتوسطات
  - الفائز

---

### 1.7 سوق الخدمات (Services Marketplace)

**صفحة القائمة:**
- بحث نصي
- فلتر الفئات (17 فئة): بناء، تشطيبات، كهرباء، تكييف، دهانات، أرضيات، مطابخ، مصاعد، نظافة، تصميم داخلي، تصميم معماري، تقييم عقاري، إدارة عقارات، تصوير، صيانة، مقاولات، مواد بناء
- فلتر المدينة
- بطاقة مزود الخدمة: شعار / اسم / تصنيف / مدينة / تقييم بالنجوم + عدد التقييمات / سعر الانطلاق / شارة "موثّق"
- زر "أضف خدمتك"

**صفحة ملف مزود الخدمة:**
- صورة الغلاف / صورة الشعار
- الاسم التجاري + الفئة + المدينة
- الوصف التفصيلي
- سعر الانطلاق
- موقع الويب
- معرض أعمال (Portfolio Images)
- التقييمات: متوسط النجوم + عدد + نموذج إضافة تقييم جديد
- أزرار التواصل

---

### 1.8 سوق الطلبات (Requests Marketplace)

**أنواع الطلبات الثلاثة:**
- طلب عقار (property): شراء أو استئجار عقار بمواصفات محددة
- طلب خدمة (service): تصميم / صيانة / تشطيب...
- طلب مسوّق (marketer): البحث عن مسوّق عقاري

**الفلاتر:**
- نوع الطلب (الكل / عقار / خدمة / مسوّق)
- حالة الطلب (مفتوح / مغلق / مؤرشف)
- المدينة

**بطاقة الطلب:**
- العنوان + نوع الطلب (شارة ملونة) + الحالة
- المدينة والحي
- الميزانية (من X إلى Y ريال)
- الوصف المختصر
- اسم الناشر + التاريخ
- أزرار: اتصال / واتساب / بريد إلكتروني

**نموذج إضافة طلب:**
- عنوان الطلب
- نوع الطلب
- الفئة (للخدمات)
- المدينة والحي
- الميزانية (حد أدنى / أقصى)
- التفاصيل
- طريقة التواصل المفضلة
- بيانات الاتصال

---

### 1.9 دليل المسوّقين (Marketers Directory)

**صفحة القائمة:**
- بحث بالاسم أو المدينة
- فلتر المدينة
- فلتر التخصص (12 تخصص)
- بطاقة المسوّق: صورة غلاف + صورة شخصية + اسم + مكتب + مدينة + تخصصات + سنوات خبرة + عدد الإعلانات النشطة + شارة "موثّق"
- أزرار: اتصال / واتساب

**صفحة ملف المسوّق:**
- صورة الغلاف + صورة شخصية
- الاسم الكامل + اسم المكتب + شارة التحقق
- المدينة + المناطق التي يخدمها
- التخصصات (tags)
- سنوات الخبرة
- نبذة تعريفية (Bio)
- إعلانات المسوّق النشطة (grid)
- التقييمات والمراجعات

### 1.10 لوحة المستخدم (Dashboard)

**4 تبويبات:**

**1 — نظرة عامة:**
- إجمالي إعلاناتي / المفضلة / الطلبات
- آخر نشاط

**2 — إعلاناتي:**
- قائمة إعلاناتي مع حالة كل إعلان
- إجراءات: عرض / تعديل / حذف / تغيير الحالة (مباع/مؤجر/أرشفة)
- زر "نشر إعلان جديد"

**3 — المفضلة:**
- العقارات المحفوظة بشبكة بطاقات
- إزالة من المفضلة

**4 — طلباتي:**
- الطلبات التي نشرتها مع حالتها
- إجراءات: تعديل / حذف / فتح-إغلاق

### 1.11 نشر إعلان جديد (Listing Form)

**الأقسام التفصيلية:**

**القسم 1 — المعلومات الأساسية:**
- عنوان الإعلان (نص)
- نوع العقار (grouped: سكني / تجاري / صناعي...)
- نوع الصفقة (بيع / إيجار / إيجار شهري / استثمار / مزاد)
- الغرض (سكني / تجاري / صناعي / زراعي / استثماري)

**القسم 2 — الموقع:**
- المنطقة الإدارية (cascading dropdown)
- المحافظة
- الحي
- العنوان التفصيلي (نص)
- خريطة اختيار الموقع (LocationPicker) مع:
  - تحديد الموقع تلقائياً عبر GPS
  - Nominatim reverse geocoding
  - دبوس قابل للسحب

**القسم 3 — التفاصيل التقنية:**
- السعر (ريال سعودي)
- المساحة الإجمالية (م²)
- المساحة المبنية (م²)
- عدد غرف النوم
- عدد دورات المياه
- عدد الأدوار
- رقم الطابق
- الواجهة (16 اتجاه)
- الأثاث (مفروش / غير / نصف)
- جودة البناء (فاخر / ممتاز / جيد / متوسط / يحتاج تجديد)
- التشطيب (كامل / شبه مشطّب / هيكل / مشطّب فاخر)
- سنة البناء
- عمر المبنى
- الطول × العرض (م)

**القسم 4 — حالة الصك:**
- قوائم جاهزة: صك إلكتروني مستقل / صك ورقي / صك مشترك / تحت الإفراز / قيد التسجيل / وقف
- أو إدخال نص حر

**القسم 5 — المزايا (Checkboxes):**
موقف سيارات، مصعد، حديقة، سطح، مسبح، غرفة خادمة، غرفة سائق، مطبخ، تكييف، عداد كهرباء، عداد ماء، مستودع، شرفة، قبو، منزل ذكي، نظام أمني، إنترنت، صرف صحي، قابل للرهن، قابل للتفاوض

**القسم 6 — القرب من الخدمات:**
مدارس، مستشفيات، مساجد، مراكز تسوق، مواصلات، حدائق، طرق رئيسية

**القسم 7 — الوسوم الخاصة:**
عاجل، حصري، مالك مباشر (بدون وسيط)

**القسم 8 — الوصف:**
محرر نص حر متعدد الأسطر

**القسم 9 — الوسائط:**
- رفع صور متعددة (drag & drop + preview + reorder)
- رفع فيديو (اختياري)

**القسم 10 — بيانات المعلن:**
- الاسم الكامل
- رقم الجوال
- البريد الإلكتروني

## القسم الثاني: التصميم شاشة بشاشة (Mobile-First)

---

### الشاشة 1 — الرئيسية (Home)

**الهيكل العام:** ScrollView عمودي بلا تبويبات (كل الأقسام تحت بعض)

**رأس الشاشة (Static Header — غير قابل للتمرير):**
- ارتفاع: 56dp
- يسار: زر الإشعارات (bell icon) + نقطة حمراء
- مركز: شعار نصي "عقار إنسايت" + أيقونة منزل صغيرة
- يمين: صورة بروفايل صغيرة دائرية (تفتح الحساب)

**قسم الهيرو (Hero):**
- ارتفاع: 520dp
- خلفية: صورة معمارية + تدرّج داكن
- نص رئيسي كبير (32sp bold): "اعثر على عقارك المثالي"
- نص ثانوي (14sp): "تصفح آلاف الإعلانات في كل مناطق المملكة"
- **لوحة البحث المدمجة (Search Panel):**
  - خلفية بيضاء شبه شفافة، cornerRadius: 20dp
  - Tabs: الكل / للبيع / للإيجار
  - مربع بحث نصي (RTL)
  - Chips أفقية لأنواع العقارات (الكل / شقة / فيلا / أرض / ...)
  - زر "بحث" كامل العرض

## القسم الخامس: تحسينات UX للجوال

---

### 5.1 نقاط ضعف الويب عند التحويل للجوال

| المشكلة في الويب | الحل في الجوال |
|---|---|
| الفلاتر المتقدمة في sidebar عريض | Bottom Sheet منزلق، فلاتر سريعة chips |
| نموذج الإضافة طويل (600+ سطر) | تجزيء إلى 6 خطوات مع شريط تقدم |
| Cascading dropdowns تعتمد على hover | Pickers أو modals للاختيار |
| الخريطة تحتاج hover لمعاينة | بطاقة تطلع من الأسفل عند الضغط |
| التحليلات بـ 5 tabs تعرض معلومات كثيفة | كل tab يعرض محتوى بسيطاً مع scroll |
| مقارنة الأحياء معقدة | تدفق 2 خطوة: اختر حيّين → شاهد المقارنة |
| عناصر hover effect كثيرة | استبدال بـ Long Press أو swipe actions |
| صفحة السجلات بجدول عريض | بطاقات عمودية مضغوطة |

---

### 5.2 تحسينات التنقل

**قاعدة "الإبهام" (Thumb Zone):**
- جميع الإجراءات الرئيسية في الثلث السفلي من الشاشة
- شريط التنقل السفلي للوصول الفوري
- زر الرجوع في أسفل يمين (مسافة متناولة)

**التنقل السياقي:**
- الضغط على الحي → فتح صفحة الحي مباشرة
- الضغط على اسم المسوّق → فتح ملف المسوّق
- Swipe left على بطاقة إعلاني → إجراءات سريعة (تعديل / حذف)

---

### 5.3 تحسينات البحث

**البحث التدريجي:**
- نتائج فورية أثناء الكتابة (debounce 300ms)
- سجل عمليات البحث الأخيرة
- مقترحات حي / مدينة (auto-complete)

**الفلاتر الذكية:**
- إبراز عدد الفلاتر الفعّالة على زر الفلاتر
- زر "مسح الكل" فوري
- حفظ الفلاتر عبر الجلسات

---

### 5.4 تحسينات الأداء

**تحميل متسلسل (Progressive Loading):**
- Skeleton UI أثناء التحميل
- اللود أولاً للمحتوى المرئي ثم البيانات التحليلية
- Lazy loading للصور

**Infinite Scroll:**
- "عرض المزيد" يُحمّل الـ batch التالي دون رفض scroll position
- Pagination كبديل احتياطي

**التخزين المؤقت:**
- الرئيسية: staleTime 10 دقائق
- التفاصيل: staleTime 5 دقائق
- التحليلات: staleTime 10 دقائق

---

### 5.5 تحسينات إمكانية الوصول

- حجم الخط الأدنى: 10sp
- حجم عناصر اللمس الأدنى: 44×44dp
- تباين الألوان: نسبة 4.5:1 على الأقل
- رسائل خطأ واضحة ومحددة
- مؤشرات تحميل في كل عملية

---

### 5.6 تدفق المستخدم المحسّن (UX Flows)

**تدفق "اشترِ عقاراً":**
```
الرئيسية → بحث بالحي/المدينة → قائمة العقارات →
فلتر متقدم → تفاصيل العقار → تقييم السعر →
اتصال بالمسوّق → (اختياري) إضافة للمفضلة
```

**تدفق "انشر إعلانك":**
```
الحساب / زر FAB → تسجيل الدخول (إن لم يكن) →
خطوة 1: النوع → خطوة 2: الموقع → خطوة 3: التفاصيل →
خطوة 4: المزايا → خطوة 5: الوسائط → خطوة 6: المراجعة والنشر
```

**تدفق "قارن الأحياء":**
```
الرئيسية → "الأحياء" → اختر حي A → اختر حي B →
شاشة المقارنة → (اختياري) تصفح عقارات الحي
```
