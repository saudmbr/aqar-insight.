Create a dedicated admin-only section called:

"التقارير وأداء المنصة"

Goal:
Give the admin a private dashboard that provides a complete reporting and performance view of the entire platform in one place.

Requirements:

1) Access control
- This section must be visible ONLY to admin accounts
- Normal users, marketers, service providers, and other roles must not see it
- Add it as a separate item in the admin area / sidebar:
  "التقارير وأداء المنصة"

2) Main purpose
This page should show the admin a complete overview of all important platform activity, performance, and growth using real internal platform data only.

3) Sections inside the page
Create the page with these main sections:

A) نظرة عامة على المنصة
B) تقارير المستخدمين
C) تقارير العقارات
D) تقارير الطلبات
E) تقارير الخدمات
F) تقارير السوق والتحليلات
G) الأداء التشغيلي
H) التقارير المالية (if platform data exists)
I) التنبيهات والمشكلات

Show top KPI cards such as:
- إجمالي المستخدمين
- إجمالي العقارات
- إجمالي الطلبات

- إجمالي الإعلانات النشطة

- عدد الإعلانات الجديدة
- عدد الطلبات الجديدة
- عدد الخدمات الجديدة

- اليوم
- الأسبوع
- الشهر
- ربع سنة
- السنة

Use professional admin analytics visuals:
- KPI cards
- line charts
- bar charts
- pie/donut charts
- tables where needed

Keep the page clean, premium, and highly readable in Arabic RTL.

14) Filters
Add filters across the reports page:
- المدة الزمنية
- المدينة
- نوع العقار
- نوع المستخدم
Apply filters consistently where relevant

15) Data rules
- Use only real internal platform data
- Do NOT show fake placeholder numbers as real admin insights
- If a metric has no data, show a truthful empty state
- Do NOT generate unsupported conclusions

for key reports

- Protect all report endpoints on the backend
- Do not expose admin reports to any other role
