Add a complete “Real Estate Marketers” module to the platform.

Goal:
Create a dedicated experience for real estate marketers so they can publish, organize, and showcase all their listings in a professional way, while allowing customers to browse all listings by each marketer as a trusted reference library.

Requirements:

1) New user role:
- Add a new role called: "real_estate_marketer"
- This role should have its own dashboard and permissions
- Marketers can create and manage their own property listings
- Normal users cannot access marketer-only controls
- Admin can manage all marketers and their listings
