| **الخط** | Cairo (Google Fonts) |

## 4. صفحات الموقع — تفاصيل كاملة صفحة بصفحة

### 4.1 الصفحة الرئيسية `/`

**أقسام الصفحة من الأعلى إلى الأسفل:**

#### أ) قسم Hero (الرئيسي)
- خلفية: صورة سينمائية + تدرج نيفي
- شعار المنصة + وصف مختصر
- **شريط بحث سريع** يحتوي على 5 فلاتر:
  - المنطقة (13 منطقة سعودية)
  - المحافظة/المدينة (تُحدّث تلقائياً عند اختيار المنطقة)
  - الحي (يُحدّث تلقائياً عند اختيار المدينة — مع حقل بحث نصي)
  - نوع العقار (شقة / فيلا / أرض / دور / مبنى / استراحة / مزرعة / محل / مكتب / مستودع)
  - نوع الصفقة (بيع / إيجار / تقسيط / استثمار / مزاد)
- زر "بحث" يوجّه إلى صفحة `/listings` مع الفلاتر

#### ب) فئات سريعة (Pills)
- أزرار نوع العقار لتصفية الإعلانات مباشرة في الصفحة
- عند الضغط: تُعرض الإعلانات المطابقة للنوع دون مغادرة الصفحة

#### ج) قسم الإعلانات
- العنوان يتغيّر حسب الحالة: "أحدث العقارات" / "الإعلانات المطابقة لبحثك" / "إعلانات [نوع]"
- يعرض آخر 8 إعلانات
- زر "عرض الكل" يوجّه إلى `/listings`

#### د) مؤشرات السوق (Analytics Panel) — قابل للتصفية
- **فلاتر التحليل:** منطقة + مدينة + حي + نوع العقار + نوع الصفقة + نطاق السعر
- **بطاقات KPI:**
  - إجمالي الإعلانات النشطة
  - متوسط السعر
  - متوسط سعر المتر المربع
  - عدد الإعلانات الجديدة (آخر 7 أيام)
  - عدد الإعلانات الجديدة (آخر 30 يوم)
- **رسم بياني خطي:** اتجاهات الأسعار عبر الزمن
- **رسم بياني دائري:** توزيع أنواع العقارات
- **جدول المناطق:** منطقة → متوسط السعر / سعر المتر / عدد الإعلانات (قابل للضغط للانتقال إلى الإعلانات)

#### هـ) قسم الخريطة التفاعلية
- خريطة Leaflet كاملة مع MarkerCluster
- **فلاتر الخريطة:** منطقة + مدينة + حي + نوع الصفقة + نوع العقار + مقياس العرض (عدد / متوسط السعر / سعر المتر)
- ألوان مختلفة: أخضر للبيع / أزرق للإيجار
- الضغط على الدبوس يُظهر: الصورة + العنوان + السعر + رابط "عرض التفاصيل"
- عداد الفلاتر النشطة

#### و) قسم مزودي الخدمات
- يعرض آخر 6 مزودي خدمات
- بطاقة لكل مزود: الصورة + الاسم + الفئة + المدينة + سعر البداية

#### ز) قسم طلبات العملاء المفتوحة
- يعرض آخر 4 طلبات مفتوحة
- بطاقة لكل طلب: العنوان + الفئة + المدينة + الميزانية

#### ح) قسم تقييم المنصة
- عدد المقيّمين + متوسط النجوم + أزرار 1-5 نجوم للتصويت

#### ط) ترحيب المستخدم / CTA للزائر
- للزائر: زر "سجّل الآن" + وصف مزايا التسجيل
- للمسجّل: رسالة ترحيبية باسمه

### 4.2 صفحة قائمة الإعلانات `/listings`

**أقسام الصفحة:**

#### أ) شريط الفلترة الكامل
| الفلتر | الوصف |
|--------|-------|
| بحث نصي | البحث في العنوان/الحي/المدينة |
| المنطقة | قائمة 13 منطقة |
| المحافظة | تتغيّر حسب المنطقة |
| الحي | يُحدّث تلقائياً + بحث نصي |
| نوع العقار | قائمة منسدلة |
| نوع الصفقة | بيع / إيجار / تقسيط / استثمار / مزاد |
| السعر من | حقل رقمي |
| غرف النوم | 1 / 2 / 3 / 4 / 5+ |

- عداد الفلاتر النشطة
- زر "مسح الفلاتر"
- الترتيب: الأحدث / الأعلى سعراً / الأوطأ سعراً

#### ب) قائمة النتائج
- بطاقات الإعلانات بعرض شبكي
- كل بطاقة تعرض: صورة + شارات (مميز/عاجل) + النوع + عنوان + السعر + الموقع + المساحة + غرف النوم/المياه
- رسالة عند عدم وجود نتائج مع زر "مسح الفلاتر"

### 4.3 صفحة تفاصيل الإعلان `/listings/:id`

#### أ) معرض الصور
- عرض أفقي للصور مع التنقل (يسار/يمين)
- مصغّرات الصور في الأسفل

#### ب) المعلومات الرئيسية
- العنوان + الموقع (منطقة/مدينة/حي)
- السعر + سعر المتر
- شارات: مميز / عاجل / حصري / موثّق / مالك مباشر
- عدد المشاهدات

#### ج) أزرار التفاعل الرئيسية
- ❤️ إضافة/إزالة من المفضلة (للمستخدم المسجّل)
- 🔗 مشاركة الإعلان
- 🚩 الإبلاغ عن الإعلان

#### د) مواصفات العقار (ألواح منظّمة)
- **أساسية:** النوع + الغرض + المساحة + الغرف + دورات المياه + المجالس + المطابخ
- **إضافية:** الطابق + عدد الطوابق + العمر + الواجهة + الأثاث + عرض الشارع + عدد الشوارع
- **المرافق الداخلية** (قائمة أيقونات)
- **المرافق القريبة** (قائمة أيقونات)
- **الوثائق:** حالة الصك + حالة الترخيص + صلاحية الرهن + رقم المرجع
- **ملاحظات البائع** (إذا وُجدت)

#### هـ) الموقع على الخريطة
- خريطة Leaflet تعرض موقع العقار الدقيق
- دبوس موقّع باسم الحي

#### و) تقييم السعر (Market Benchmark) — ذكاء سوقي
- **مقارنة مع الحي:** سعر العقار vs متوسط الحي → نسبة الفرق % + وصف (مناسب/مرتفع/منخفض)
- **مقارنة مع المدينة:** سعر العقار vs متوسط المدينة → نسبة الفرق %
- **مقارنة مع النوع:** سعر العقار vs متوسط نفس النوع → نسبة الفرق %
- رسالة "بيانات غير كافية" عند قلة الإعلانات المشابهة

#### ز) ملف المسوّق (إذا كان الإعلان مرتبطاً بمسوّق)
- صورة المسوّق + الاسم + اسم المكتب
- تقييم المسوّق (نجوم)
- أزرار: واتساب المسوّق + عرض ملفه الكامل

#### ح) بطاقة التواصل الثابتة
- زر واتساب (يفتح محادثة جاهزة بعنوان الإعلان)
- زر الاتصال الهاتفي

#### ط) إعلانات مشابهة
- 4-6 إعلانات من نفس المنطقة/النوع

### 4.5 صفحة التحليلات `/analytics`

**5 تبويبات:**

|---------|---------|
| **A — مؤشر السوق** | نقاط السوق (0-100) + تصنيف حالة السوق + 6 مؤشرات: متوسط السعر / سعر المتر / إجمالي الإعلانات / عدد للبيع / عدد للإيجار / الجديد في 30 يوم |
| **B — الأسعار** | رسم بياني خطي لاتجاهات الأسعار + فلتر الفترة الزمنية: يومي / أسبوعي / شهري / ربع سنوي / سنوي |

| **E — الذكاء التحليلي** | رؤى تلقائية مولّدة + **توصية ذكية** من أربعة خيارات: 🛒 شراء الآن / ⏳ انتظر / ⚖️ فرصة تفاوض / ➖ مستقر |

**فلاتر مشتركة لجميع التبويبات:**

### 4.15 صفحة إعداداتالحساب `/account`

**الأقسام:**

## 6. التحليلات — التفاصيل الكاملة

### 6.1 تقييم السعر (Benchmark) — لكل إعلان
- **مقارنة مع الحي:** يجلب متوسط سعر المتر لنفس الحي → يحسب الفرق % → يصنّف: مناسب / أعلى من المتوسط / أقل من المتوسط
- **مقارنة مع المدينة:** نفس الآلية لمستوى المدينة
- **مقارنة مع النوع:** مقارنة بمتوسط نفس نوع العقار
- يعرض عدد الإعلانات المستخدمة في الحساب للمصداقية

### 6.2 مؤشر صحة السوق (Market Health Score)
- درجة من 0-100
- توصية تلقائية: 🛒 شراء الآن / ⏳ انتظر / ⚖️ فرصة تفاوض / ➖ مستقر
- يعتمد على: معدل الدوران + التوازن بين العرض والطلب + اتجاه الأسعار

### 6.3 مؤشرات الأحياء
- **عقد** (مرتفع جداً): أسعار المنطقة أعلى بكثير من المتوسط
- **فرصة** (منخفض): أسعار أقل من المتوسط بشكل ملحوظ
- **مراقبة** (متوسط): ضمن النطاق الطبيعي
