Build a centralized internal analytics system for the platform.

Goal:
Create a robust Analytics Engine that automatically calculates real estate market indicators using ONLY internal platform data, then connects these indicators consistently across the dashboard, map, search, neighborhood pages, and listing pages.

==================================================
1) CENTRAL ANALYTICS ENGINE
==================================================

Create a centralized Analytics Engine responsible for calculating all market indicators automatically.

This engine must depend ONLY on internal platform data:
- listing data
- prices
- listing created date
- listing updated date
- listing status
- location (city, district/neighborhood)
- property type
- listing purpose (sale / rent if available)

Requirements:
- one central logic source for all indicators
- reusable across all pages
- no duplicated indicator logic in multiple components
- scalable and maintainable
- all outputs must be explainable and data-driven
